## Features
 - Start page
 - Introduction
 - Location
 - Inside
 - Activities
 - Booking
 - Seasonal activities
 - Footer
 
### Other features
 - Multi language
 - Rose graph (to visualize distance and activity)
 - Booking service frontend with re-captcha integration
 
## Usage

Install dependencies:
```
npm install
```

Start the website:
```
ng serve
```
